В проекте представлено выполнение тестового задания для компании Azur Games. Используемая версия Unity 2022.3.62f2
Префаб свёрстанного экрана - Hero Screen. Он также добавлен на стартовую сцену для удобства просмотра
При запуске плеймода стартуют анимации появления интерфейса, схожие с оригинальными анимациями из игры WWII Defense. Также реализованы анимациии нажатия на кнопки
